# Fonts folder

## Louvre (title font)

The **Louvre** font is used for the main title "Simsiqueste".

The original link (gfxfile.com/gfi/Louvre.zip) returns 404. Download Louvre from one of these instead:

- https://www.1001fonts.com/louvre-font.html (free for commercial use)
- https://www.fontmirror.com/louvre
- https://www.allfreefonts.co/louvre-font/

Then place **one** of these files in this folder:

- `Louvre.otf`, or
- `Louvre.ttf`

The site will use whichever file is present.
